# uc_pyircbot
A python IRC bot

## Setting up
```shell
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
    
## Running
```shell
cd bot
errbot
```

## Docker repo
In order to deploy this on our infrastructure, pushes to master trigger a build on [docker hub](https://hub.docker.com/r/fossgalaxy/errbot/)
